# Car Price Prediction 🚗

This project predicts car selling prices using Machine Learning.  
It is part of the **CodeAlpha Data Science Internship**.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
The dataset contains car details such as year of manufacture, fuel type, transmission, kilometers driven, and present price.

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Train-Test Split
6. Linear Regression Model Training
7. Model Evaluation

## Machine Learning Model
Linear Regression

## Result
The model predicts car selling prices based on various car features with reasonable accuracy.
